Descrição:
Este programa visa a implementação do jogo Rummikub.
A linguagem de programação utilizada foi C e o programa consiste em jogar Rummikub seguindo as regras do jogo. Os números são de 1 a 9 e de A a D, seguindo o padrão tetradecimal, mas retirando o zero, e as cores são representadas pelos símbolos "!", "@", "#" e "$". Há no programa funções para criar grupos para a mesa e colocar cartas da sua mão nesses grupos, assim como, poder manipulá-los. Além dessas, também há funções que garantem o funcionamento do jogo e atentam as regras, tal como uma que checa se a sequência de cartas feita pelo jogador é válida. 

Compilar e executar:
Linux -> Com o terminal aberto, acesse o diretório em que está localizado o programa. Depois, digite: gcc Trab_4.c -o <nome desejado>.out.
Uma vez confirmada a compilação, execute-o com o comando: ./<nome desejado>.out

Windows -> Para a compilação e execucão do programa, pode-se usar o auxílio de IDEs como o Code:Blocks. Abra o arquivo Trab_4.c na IDE. Compile e execute sem paramêtros.
